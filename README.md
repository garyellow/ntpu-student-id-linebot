# ntpu-student-id-linebot

a line bot about student id

## 使用方式

1. 輸入**學號**獲取**學生姓名**
2. 輸入**系名**獲取**系代碼**
3. 輸入**系代碼**獲取**系名**
4. 輸入**入學學年**並依序**選取科系**獲取**某學年度某系的學生名單**
